# Mercado Livre's frontend practical test (backend node)

Encapsulate MercadoLivre's APIs and map the responses for the front-end application.

## How to run the app:

- First of all, clone this repo, navigate into root directory and run: `yarn`

- Having finished the dependency downloads, you can use the following commands:

    - To run the app locally: `yarn start`
    - To build: `yarn run build`

This application runs at port 4000. You can change this editting the `.env` file.
